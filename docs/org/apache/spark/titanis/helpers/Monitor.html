Monitor

class Monitor extends Configurable

Helper class used to record processing times for the prediction, evaluation, and training batch queries for each batch and write the results of the batch queries to a CSV file.

The processing times for each phase of the batch query are stored in the metricsRow before it is added to a DataFrame and written out to a CSV file. Each phase of the query has a corresponding method to add the processing time recorded to the metricsRow.

The updateMetrics and writeMetrics method write the processing time DataFrame to a pre-determined CSV file.

Each phase of the query has a corresponding method to add the resultant DataFrame at the end of the phase to a CSV file.
