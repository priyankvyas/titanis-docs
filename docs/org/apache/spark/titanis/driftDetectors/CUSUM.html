CUSUM

class CUSUM extends DriftDetector

The drift detector that uses the CUSUM drift detection method to detect concept drift.

The false alarm rate can be configured using the false alarm rate option.

The drift detector has global variables to retain the incremental sums from the previous batch. The accumulators are used to maintain the incremental sums while processing the batch. For each of the required incremental sums, a User Defined Function is used to update the accumulator and correct the sum with the previous batch's results when required. The initOverride method initializes all the required variables and accumulators. The detect method initiates the detection streaming query that adds the necessary columns and outputs the stream with the detectedChange column.
