  •  package titanis

    Titanis is a machine learning and data analysis platform tailored to optimize traditional machine learning algorithms for streaming input data, leveraging a distributed system architecture.

    Titanis is a machine learning and data analysis platform tailored to optimize traditional machine learning algorithms for streaming input data, leveraging a distributed system architecture. runTask is the primary entry point of the application. Titanis has native implementations of kNN and Hoeffding Tree. It can also utilize implementations of MOA learners from the MOA library.

    More native learner and helper implementations are to be added in future versions.

  1.  class DirectoryReader extends StreamReader

    Reader used to read batch files from a directory as a stream.

    Reader used to read batch files from a directory as a stream.

    The reader requires the path to the directory that will receive the data as a stream. The directory will be observed and all the files that are present in the directory will be treated as separate batches of a stream. It will read the batches in order of their arrival into the directory.

    Current version of the directory reader only supports CSV files for the input data. Future versions will allow for data to be represented in different formats.

    Since

    0.0.1

  2.  trait StreamReader extends Configurable

    Defines a Stream Reader.

    Defines a Stream Reader.

    A Stream Reader reads the data from the input data stream to prepare it for pre-processing for the learner. A Stream Reader must contain a schemaFileNameOption that is used to provide the reader the path to the header file, a getData method that parses the data from the input source, and a getSchema method that parses the schema of the data stream from the input header file. Stream Readers would mainly differ in terms of which input source they can read from.

    The Stream Reader stores the schema of the input data stream in the schema member variable.

    Used by DirectoryReader.

    Since

    0.0.1
