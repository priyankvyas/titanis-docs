  •  package titanis

    Titanis is a machine learning and data analysis platform tailored to optimize traditional machine learning algorithms for streaming input data, leveraging a distributed system architecture.

    Titanis is a machine learning and data analysis platform tailored to optimize traditional machine learning algorithms for streaming input data, leveraging a distributed system architecture. runTask is the primary entry point of the application. Titanis has native implementations of kNN and Hoeffding Tree. It can also utilize implementations of MOA learners from the MOA library.

    More native learner and helper implementations are to be added in future versions.

  1.  class Monitor extends Configurable

    Helper class used to record processing times for the prediction, evaluation, and training batch queries for each batch and write the results of the batch queries to a CSV file.

    Helper class used to record processing times for the prediction, evaluation, and training batch queries for each batch and write the results of the batch queries to a CSV file.

    The processing times for each phase of the batch query are stored in the metricsRow before it is added to a DataFrame and written out to a CSV file. Each phase of the query has a corresponding method to add the processing time recorded to the metricsRow.

    The updateMetrics and writeMetrics method write the processing time DataFrame to a pre-determined CSV file.

    Each phase of the query has a corresponding method to add the resultant DataFrame at the end of the phase to a CSV file.

    Since

    0.0.1

  2.  class ShutdownListener extends StreamingQueryListener with Logging

    Streaming Query Listener for the training streaming query that is used to terminate all the streaming queries and the application.

    Streaming Query Listener for the training streaming query that is used to terminate all the streaming queries and the application.

    The ShutdownListener overrides the onQueryStarted, onQueryProgress, and onQueryTerminated methods inherited from the Streaming Query Listener abstract class.

    It uses a Long counter and threshold to determine when the Spark application should be terminated.
