  •  package titanis

    Titanis is a machine learning and data analysis platform tailored to optimize traditional machine learning algorithms for streaming input data, leveraging a distributed system architecture.

    Titanis is a machine learning and data analysis platform tailored to optimize traditional machine learning algorithms for streaming input data, leveraging a distributed system architecture. runTask is the primary entry point of the application. Titanis has native implementations of kNN and Hoeffding Tree. It can also utilize implementations of MOA learners from the MOA library.

    More native learner and helper implementations are to be added in future versions.

  1.  trait Learner extends Configurable with HasFeaturesCol with HasLabelCol with HasPredictionCol with HasWeightCol with HasIDCol

    Defines a Learner.

    Defines a Learner.

    A Learner must contain an init method, a predict method, an evaluate method, and a train method. Learners differ in the way they are initialized, make predictions, are evaluated, and are trained. The Learner initializes streaming queries for the prediction, evaluation, and training processes. It must contain the additional override methods for the learner specific initialization, prediction, and training processes. These methods are initOverride, predictOverride, and trainOverride, respectively.

    A Learner has three flags that are triggered when the corresponding streaming queries finish processing a batch of the input data. This is to indicate to the next stage of the process to begin.

    The Learner stores the schema of the input data stream in the schema member variable.

    The Learner stores the number of classes present in the input data stream in the numClasses member variable. This is used only for data that has a nominal target attribute. The type of the target attribute is stored in the targetType member variable. It can either be "numeric" or "nominal".

    The column names of all the predictor attributes in the input data stream are stored in a collection in the predictorAttributes member variable.

    A Learner has a Monitor that is initialized in the init method. The Monitor records the processing duration of the streaming queries and writes the resulting DataFrames to a CSV file.

    A Learner overrides the copy method inherited from the Params traits. It returns a default copy of the Learner object with extra parameters.

    Used by org.apache.spark.titanis.learners.`lazy`.LazyLearner, org.apache.spark.titanis.learners.ensemble.Ensemble, and org.apache.spark.titanis.learners.MOA.MOALearner.
