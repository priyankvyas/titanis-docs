package titanis

Titanis is a machine learning and data analysis platform tailored to optimize traditional machine learning algorithms for streaming input data, leveraging a distributed system architecture. runTask is the primary entry point of the application. Titanis has native implementations of kNN and Hoeffding Tree. It can also utilize implementations of MOA learners from the MOA library.

More native learner and helper implementations are to be added in future versions.

  1.  object runTask extends Serializable with Logging

    Executor of org.apache.spark.titanis.tasks.Task instances.

    Executor of org.apache.spark.titanis.tasks.Task instances.

    The primary entry point for the Titanis application. RunTask, as the name suggests, runs the task provided in the spark-submit command as an application argument. It should be specified as the class option value in the spark-submit command. This is to signal to Spark that this class is the main class for the application.

    Example:

    1. spark-submit --class "org.apache.spark.titanis.runTask" ".\target\scala-2.13\titanis_2.13-0.1.jar" <application arguments>
