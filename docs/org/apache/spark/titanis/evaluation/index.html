  •  package titanis

    Titanis is a machine learning and data analysis platform tailored to optimize traditional machine learning algorithms for streaming input data, leveraging a distributed system architecture.

    Titanis is a machine learning and data analysis platform tailored to optimize traditional machine learning algorithms for streaming input data, leveraging a distributed system architecture. runTask is the primary entry point of the application. Titanis has native implementations of kNN and Hoeffding Tree. It can also utilize implementations of MOA learners from the MOA library.

    More native learner and helper implementations are to be added in future versions.

  1.  class BasicClassificationEvaluator extends Evaluator with Logging

    The evaluator that is used to evaluate the classification performance of learners.

    The evaluator that is used to evaluate the classification performance of learners.

    The evaluator is configurable to include per-class metrics, confusion matrix, micro-averages, macro-averages, and weighted-averages using corresponding flags. The beta for the f-beta score can be set using the beta option.

    The addResult method is used to evaluate the predictions made by the learner. The computeMatrix method is used to count the number of true positive, true negative, false positive, and false negative predictions. The calculateMetricsBinary and calculateMetricsMulticlass method calculates the recall, precision, accuracy, and other classification metrics.

    Since

    0.0.1

  2.  class BasicRegressionEvaluator extends Evaluator

    The evaluator that is used to evaluate the regression performance of learners.

    The evaluator that is used to evaluate the regression performance of learners.

    The addResult method is used to evaluate the predictions made by the learner. The calculateMetricsRegression method calculates the Mean Absolute Error, Root Mean Squared Error, Mean Squared Error, R squared and Adjusted R squared score as the regression metrics.

  3.  trait Evaluator extends Configurable

    Defines an Evaluator.

    Defines an Evaluator.

    An Evaluator must contain the init method that initializes and populates the required member variables for the Evaluator and the addResult method that performs the necessary calculations for the evaluation metrics for the learning task. Evaluators differ in terms of the metrics they use for evaluation, which in turn, depends on the learning task.

    An Evaluator stores the schema of the input data in the schema member variable.

    If the results of the first batch of data are to be included in evaluation, they can be included using the corresponding option. If cumulative results for the micro-batches are not required, batchwise results can be obtained using the corresponding option.

    An Evaluator is initialized using the schema of the input data stream.

    Used by BasicClassificationEvaluator and BasicRegressionEvaluator.

    Since

    0.0.1
