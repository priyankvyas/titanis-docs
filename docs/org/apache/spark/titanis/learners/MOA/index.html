  •  package titanis

    Titanis is a machine learning and data analysis platform tailored to optimize traditional machine learning algorithms for streaming input data, leveraging a distributed system architecture.

    Titanis is a machine learning and data analysis platform tailored to optimize traditional machine learning algorithms for streaming input data, leveraging a distributed system architecture. runTask is the primary entry point of the application. Titanis has native implementations of kNN and Hoeffding Tree. It can also utilize implementations of MOA learners from the MOA library.

    More native learner and helper implementations are to be added in future versions.
