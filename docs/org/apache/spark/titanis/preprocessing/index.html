  •  package titanis

    Titanis is a machine learning and data analysis platform tailored to optimize traditional machine learning algorithms for streaming input data, leveraging a distributed system architecture.

    Titanis is a machine learning and data analysis platform tailored to optimize traditional machine learning algorithms for streaming input data, leveraging a distributed system architecture. runTask is the primary entry point of the application. Titanis has native implementations of kNN and Hoeffding Tree. It can also utilize implementations of MOA learners from the MOA library.

    More native learner and helper implementations are to be added in future versions.

  2.  class StreamPrepareForLearning extends StreamPreprocessor

    Preprocessor used to prepare the streaming DataFrame for the learning stage(s).

    Preprocessor used to prepare the streaming DataFrame for the learning stage(s).

    The target and weight attribute identifiers should be given, if those attributes are named something else.

    StreamPrepareForLearning is used to map nominal features to distinct indices to allow the learners to use numeric values to represent the nominal values.

    The preprocessor adds a row count to the stream to allow for indexing and partitioning the streaming DataFrame evenly across all executors, based on the indices. This should serve as a count for the number of input rows received in a batch as well.

    Since

    0.0.1
